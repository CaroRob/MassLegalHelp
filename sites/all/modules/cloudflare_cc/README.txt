INTRODUCTION
------------

This module allows a user to clear their Cloudflare Cache whenever Drupal's
"Clear Cache" is called.

 * For a full description of the module, visit the project page:
   https://drupal.org/project/cloudflare_cc

 * To submit bug reports and feature suggestions, or to track changes:
   https://drupal.org/project/issues/cloudflare_cc

REQUIREMENTS
------------

This module requires that you have a Cloudflare account. You should also make
sure that that Cloudflare account is wired to your web site.

RECOMMENDED MODULES
-------------------

 None

INSTALLATION
------------

 * Install as you would normally install a contributed Drupal module. See:
   https://drupal.org/documentation/install/modules-themes/modules-7
   for further information.

CONFIGURATION
-------------

 * Visit the configuration page under Admin > Configuration > Web Services

TROUBLESHOOTING
---------------

 * If the cache does not clear after doing a Drupal Cache Clear:

   - Did you enter your credentials correctly?

   - Did you actually clear the cache?
